# Snakeware
Snakeware is an app runtime.

## Running Snakeware
### Windows + Linux

1. Clone the repo: `git clone https://github.com/TheRealInkDevil/snakeware.git`
2. Run `sw.py`

## Dependencies
Snakeware only requires a functioning installation of Python.
